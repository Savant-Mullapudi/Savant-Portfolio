# Savant Mullapudi — Personal Portfolio

A personal portfolio website designed to showcase my work, experience, and projects as a Full Stack Engineer. Built with vanilla HTML, CSS, and JavaScript — clean and fast front-end code.

**Live Site**: [savant-mullapudi.github.io/Savant-Portfolio](https://savant-mullapudi.github.io/Savant-Portfolio/)

---

## Features

- **Responsive design** — fully functional across desktop, tablet, and mobile
- **Smooth navigation** — single-page layout with anchor-based section scrolling
- **Interactive tech stack** — clickable icons that reveal details about each tool
- **Experience timeline** — structured walkthrough of roles with expandable company cards
- **Projects gallery** — curated showcase of personal and professional work with live links
- **Contact section** — direct links to email, phone, LinkedIn, and GitHub

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, flexbox, media queries) |
| Interactivity | Vanilla JavaScript (ES6+) |
| Hosting | GitHub Pages |
| Design | Figma (wireframing & layout planning) |

No external frameworks or libraries — lightweight for fast load times and easy maintainability.

---

## Project Structure

```
Savant-Portfolio/
├── index.html          # Main HTML file (single-page layout)
├── style.css           # Global styles and responsive breakpoints
├── mediaqueries.css    # Additional responsive rules for different screen sizes
├── script.js           # Navigation, interactivity, and scroll behavior
└── assets/             # Images, logos, and profile picture
```

---

## Running Locally

No build step required. Just clone and open:

```bash
git clone https://github.com/Savant-Mullapudi/Savant-Portfolio.git
cd Savant-Portfolio
open index.html
```

Or use a local dev server:

1. Open the project folder in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. The site will open at `http://localhost:5500`
